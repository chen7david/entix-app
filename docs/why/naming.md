# Why enforce strict naming?

Consistent naming reduces cognitive load when switching between frontend and backend.

## Benefits
1. **Searchability**: Predictable file names make it easier to find files via "Open File" shortcuts.
2. **Context**: Seeing `PascalCase` immediately signals a React component or a Class definition.
3. **Collaboration**: New developers can guess where a file belongs or how a variable is formatted without checking a guide every time.
