# Why branch protection?

Branch protection ensures quality and stability on the `main` branch.

## Benefits
1. **Accidental Deletion**: Prevents anyone from accidentally force-pushing or deleting `main`.
2. **Build Integrity**: Forces Cloudflare Pages to verify that the project can still build before merging.
3. **Collaboration**: Encourages code review through required approvals, even if they are self-approvals for solo speed.
