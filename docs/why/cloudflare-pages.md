# Why use Cloudflare Pages?

Cloudflare Pages provides a complete platform for building and hosting static and dynamic web applications.

## Benefits
1. **Edge Integration**: Native support for Cloudflare Workers (Functions) for dynamic backends.
2. **First-Class Git Support**: Automatic build/deploy on push, plus preview environments for every PR.
3. **Performance**: Static assets are cached globally on Cloudflare's CDN by default.
4. **Developer Experience**: Simple configuration, integrated logs, and built-in secret management.
